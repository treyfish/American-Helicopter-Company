# American Helicopter Company — Website Revamp Demo

Cinematic single-page demo rebuild of [americanhelicoptercompany.com](https://americanhelicoptercompany.com/) — "The American Hangar" concept.

- **Stack:** single `index.html`, Tailwind CDN, GSAP + ScrollTrigger, no build step
- **Highlights:** full-bleed hero video (recut from the client's brand film, 993 MB → 4.3 MB), interactive R22/R44/R66 fleet showcase, embedded Robinson live 3D factory configurator with build-sheet PDF → contact-form flow
- **Local dev:** `node serve.mjs` → http://localhost:3000
- **Screenshots:** `node screenshot.mjs http://localhost:3000 [label] [width] [height]` (requires `npm install`)

`reference-old-site/` holds captures of the current live site for before/after comparison. `reference-configurator/` documents Robinson's factory configurator.
